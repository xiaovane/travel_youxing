套餐说明
本产品提供皮皮岛北+竹子岛+蚊子岛一日游行程，请按照您下榻的酒店位置，选择预订

详细行程
08:00-09:30        司机到酒店接您，到达码头
您可以在码头更衣室更换泳衣，享受简单早餐（咖啡、茶点），专业中文导游为您讲解行程安排后，出发前往皮皮岛北部
10:30-12:00        抵达皮皮岛北
在这里进行第一次浮潜，海水清澈，热带小鱼很多，浮潜结束后登上皮皮岛
12:00-13:00        午餐时间
在皮皮岛上的大自然度假村享用当地泰餐
13:00-14:30        前往竹子岛
登上竹子岛，享受白沙和日光浴，海边游泳
14:30-15:30        前往蚊子岛
在蚊子岛周边海域进行第二次浮潜，这里的珊瑚很漂亮
15:30-16:00        返回码头，司机送您回酒店

景点介绍
皮皮岛北的浮潜点
皮皮岛以柔软洁白的沙滩，宁静碧蓝的海水著称，北部有一处绝佳的浮潜地点，海水平静，清澈见底，热带小鱼成群，一边浮潜一边欣赏周围天然的喀斯特地貌


大自然度假村餐厅
餐厅临海，有270度海景露台，午餐是泰式自助，有洋葱鱿鱼圈、炸海鱼、冬阴功汤、菠萝炒鸡、炸鸡腿、空心菜等当地新鲜食材做成的美味佳肴，荤素搭配，餐厅还提供当地特色的绿豆凉糕等消暑降温


餐厅内270度海景露台


美味泰式自助


竹子岛
这里沙滩细腻，不像皮皮岛是粗粒沙，海水非常清澈，但是这里小鱼不多，更适合在沙滩上休闲日光浴或在海边玩水


蚊子岛浮潜点
原生态的小岛，这里鱼群很多，可以在浮潜时带着面包或水果喂小鱼



延展阅读

普吉岛新型的高端船公司，以各种高达上的船型和优质服务出名。船只停泊码头是普吉最高档的出海码头，专业的船员和中文导游为您全程服务，sea star公司为客人提供的美味泰式午餐也是亮点之一。
